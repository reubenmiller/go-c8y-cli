Get audit record collection

### Synopsis

Audit records contain information about modifications to other Cumulocity entities. For example the audit records contain each operation state transition, so they can be used to check when an operation transitioned from PENDING -> EXECUTING -> SUCCESSFUL.


```
c8y auditrecords list [flags]
```

### Examples

```
$ c8y auditRecords list --pageSize 100
Get a list of audit records
        
```

### Options

```
      --application string   Application
      --dateFrom string      Start date or date and time of audit record occurrence.
      --dateTo string        End date or date and time of audit record occurrence.
  -h, --help                 help for list
      --revert               Return the newest instead of the oldest audit records. Must be used with dateFrom and dateTo parameters
      --source string        Source Id or object containing an .id property of the element that should be detected. i.e. AlarmID, or Operation ID. Note: Only one source can be provided (accepts pipeline)
      --type string          Type
      --user string          Username
```
