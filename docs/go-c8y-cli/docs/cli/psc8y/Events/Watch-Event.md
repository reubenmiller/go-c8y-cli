## SYNOPSIS
Watch realtime events

## SYNTAX

```
Watch-Event
	[[-Device] <Object>]
	[[-Duration] <String>]
	[[-Count] <Int32>]
	[[-ActionTypes] <String[]>]
	[<CommonParameters>]
```

## DESCRIPTION
Watch realtime events

## EXAMPLES

### EXAMPLE 1
```
Watch-Event -Device 12345
Watch all events for a device
```
