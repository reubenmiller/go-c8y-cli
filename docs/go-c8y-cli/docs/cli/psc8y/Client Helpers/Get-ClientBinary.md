## SYNOPSIS
Get the path to the Cumulocity Binary

## SYNTAX

```
Get-ClientBinary
	[<CommonParameters>]
```

## DESCRIPTION
Get the full path to the Cumulocity Binary which is compatible with the current Operating system

## EXAMPLES

### EXAMPLE 1
```
Get-ClientBinary
```

Returns the fullname of the path to the Cumulocity binary
