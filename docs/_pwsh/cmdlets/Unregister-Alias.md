# Unregister-Alias

## SYNOPSIS
Unregister aliases

## SYNTAX

```
Unregister-Alias
	[<CommonParameters>]
```

## DESCRIPTION
Unregister any aliases which were registered by the Register-Alias cmdlet

## EXAMPLES

### EXAMPLE 1
```
Unregister-Alias
```
