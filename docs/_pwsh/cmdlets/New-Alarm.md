# New-Alarm

## SYNOPSIS
Create a new alarm

## SYNTAX

```
New-Alarm
	[-Device] <Object[]>
	[-Type] <String>
	[[-Time] <String>]
	[-Text] <String>
	[-Severity] <String>
	[[-Status] <String>]
	[[-Data] <Object>]
	[-Raw]
	[[-OutputFile] <String>]
	[-NoProxy]
	[[-Session] <String>]
	[[-TimeoutSec] <Double>]
	[-Force]
	[-WhatIf]
	[-Confirm]
	[<CommonParameters>]
```

## DESCRIPTION
Create a new alarm on a device or agent.

## EXAMPLES

### EXAMPLE 1
```
New-Alarm -Device $device.id -Type c8y_TestAlarm -Time "-0s" -Text "Test alarm" -Severity MAJOR
```

Create a new alarm for device

### EXAMPLE 2
```
Get-Device -Id $device.id | PSc8y\New-Alarm -Type c8y_TestAlarm -Time "-0s" -Text "Test alarm" -Severity MAJOR
```

Create a new alarm for device (using pipeline)
