# Get-CurrentTenantApplications

## SYNOPSIS

## SYNTAX

```
Get-CurrentTenantApplications
	[<CommonParameters>]
```

## DESCRIPTION
Get the list of applications that are subscribed on the current tenant

## EXAMPLES

### EXAMPLE 1
```
Get-CurrentTenantApplications
```

Get a list of applications in the current tenant
