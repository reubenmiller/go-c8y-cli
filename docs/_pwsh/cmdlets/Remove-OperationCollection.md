# Remove-OperationCollection

## SYNOPSIS
Delete a collection of operations

## SYNTAX

```
Remove-OperationCollection
	[[-Agent] <Object[]>]
	[[-Device] <Object[]>]
	[[-DateFrom] <String>]
	[[-DateTo] <String>]
	[[-Status] <String>]
	[-Raw]
	[[-OutputFile] <String>]
	[-NoProxy]
	[[-Session] <String>]
	[[-TimeoutSec] <Double>]
	[-Force]
	[-WhatIf]
	[-Confirm]
	[<CommonParameters>]
```

## DESCRIPTION
Delete a collection of operations using a set of filter criteria.
Be careful when deleting operations.
Where possible update operations to FAILED (with a failure reason) instead of deleting them as it is easier to track.

## EXAMPLES

### EXAMPLE 1
```
Remove-OperationCollection -Device "{{ randomdevice }}" -Status PENDING
```

Remove all pending operations for a given device
