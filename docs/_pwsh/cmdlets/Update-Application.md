# Update-Application

## SYNOPSIS
Update application meta information

## SYNTAX

```
Update-Application
	[-Id] <Object[]>
	[[-Data] <Object>]
	[[-Name] <String>]
	[[-Key] <String>]
	[[-Availability] <String>]
	[[-ContextPath] <String>]
	[[-ResourcesUrl] <String>]
	[[-ResourcesUsername] <String>]
	[[-ResourcesPassword] <String>]
	[[-ExternalUrl] <String>]
	[[-Template] <String>]
	[[-TemplateVars] <String>]
	[-Raw]
	[[-OutputFile] <String>]
	[-NoProxy]
	[[-Session] <String>]
	[[-TimeoutSec] <Double>]
	[-Force]
	[-WhatIf]
	[-Confirm]
	[<CommonParameters>]
```

## DESCRIPTION
Update an application by its id

## EXAMPLES

### EXAMPLE 1
```
Update-Application -Id $App.name -Availability "MARKET"
```

Update application availability to MARKET
