# Update-Tenant

## SYNOPSIS
Update tenant

## SYNTAX

```
Update-Tenant
	[[-Id] <Object>]
	[[-Company] <String>]
	[-Domain] <String>
	[[-AdminName] <String>]
	[[-AdminPass] <String>]
	[[-ContactName] <String>]
	[[-ContactPhone] <String>]
	[[-Data] <Object>]
	[-Raw]
	[[-OutputFile] <String>]
	[-NoProxy]
	[[-Session] <String>]
	[[-TimeoutSec] <Double>]
	[-Force]
	[-WhatIf]
	[-Confirm]
	[<CommonParameters>]
```

## DESCRIPTION
Update tenant

## EXAMPLES

### EXAMPLE 1
```
Update-Tenant -Id mycompany -ContactName "John Smith"
```

Update a tenant by name (from the mangement tenant)
