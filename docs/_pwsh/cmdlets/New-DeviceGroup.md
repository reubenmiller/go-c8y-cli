# New-DeviceGroup

## SYNOPSIS
Create device group

## SYNTAX

```
New-DeviceGroup
	[-Name] <String>
	[[-Type] <String>]
	[[-Data] <Object>]
	[[-ProcessingMode] <String>]
	[[-Template] <String>]
	[[-TemplateVars] <String>]
	[-Raw]
	[[-OutputFile] <String>]
	[-NoProxy]
	[[-Session] <String>]
	[[-TimeoutSec] <Double>]
	[-Force]
	[-WhatIf]
	[-Confirm]
	[<CommonParameters>]
```

## DESCRIPTION
Create a new device group to logically group one or more devices

## EXAMPLES

### EXAMPLE 1
```
New-DeviceGroup -Name $GroupName
```

Create device group

### EXAMPLE 2
```
New-DeviceGroup -Name $GroupName -Data @{ "myValue" = @{ value1 = $true } }
```

Create device group with custom properties
