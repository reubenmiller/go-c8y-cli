# New-AuditRecord

## SYNOPSIS
Create a new audit record

## SYNTAX

```
New-AuditRecord
	[-Type] <String>
	[[-Time] <String>]
	[-Text] <String>
	[-Source] <String>
	[-Activity] <String>
	[-Severity] <String>
	[[-User] <String>]
	[[-Application] <String>]
	[[-Data] <Object>]
	[[-Template] <String>]
	[[-TemplateVars] <String>]
	[-Raw]
	[[-OutputFile] <String>]
	[-NoProxy]
	[[-Session] <String>]
	[[-TimeoutSec] <Double>]
	[-Force]
	[-WhatIf]
	[-Confirm]
	[<CommonParameters>]
```

## DESCRIPTION
Create a new audit record for a given action

## EXAMPLES

### EXAMPLE 1
```
New-AuditRecord -Type "ManagedObject" -Time "0s" -Text "Managed Object updated: my_Prop: value" -Source $Device.id -Activity "Managed Object updated" -Severity "information"
```

Create an audit record for a custom managed object update
