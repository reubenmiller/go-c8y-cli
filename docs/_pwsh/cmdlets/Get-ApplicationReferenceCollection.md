# Get-ApplicationReferenceCollection

## SYNOPSIS
Get a collection of application references on a tenant

## SYNTAX

```

	[[-Tenant] <Object>]
	[[-PageSize] <Int32>]
	[-WithTotalPages]
	[-Raw]
	[[-OutputFile] <String>]
	[-NoProxy]
	[[-Session] <String>]
	[[-TimeoutSec] <Double>]
	[-WhatIf]
	[-Confirm]
	[<CommonParameters>]
```
