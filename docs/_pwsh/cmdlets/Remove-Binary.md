# Remove-Binary

## SYNOPSIS
Delete binary

## SYNTAX

```
Remove-Binary
	[-Id] <String>
	[-Raw]
	[[-OutputFile] <String>]
	[-NoProxy]
	[[-Session] <String>]
	[[-TimeoutSec] <Double>]
	[-Force]
	[-WhatIf]
	[-Confirm]
	[<CommonParameters>]
```

## DESCRIPTION
Delete a binary from the tenant.
This delete operation can not be undone.

## EXAMPLES

### EXAMPLE 1
```
Remove-Binary -Id $Binary.id
```

Delete a binary
