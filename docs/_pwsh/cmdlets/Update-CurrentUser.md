# Update-CurrentUser

## SYNOPSIS
Update the current user

## SYNTAX

```
Update-CurrentUser
	[[-FirstName] <String>]
	[[-LastName] <String>]
	[[-Phone] <String>]
	[[-Email] <String>]
	[[-Enabled] <String>]
	[[-Password] <String>]
	[[-ProcessingMode] <String>]
	[[-Template] <String>]
	[[-TemplateVars] <String>]
	[-Raw]
	[[-OutputFile] <String>]
	[-NoProxy]
	[[-Session] <String>]
	[[-TimeoutSec] <Double>]
	[-Force]
	[-WhatIf]
	[-Confirm]
	[<CommonParameters>]
```

## DESCRIPTION
Update properties or settings of your user such as first/last name, email or password

## EXAMPLES

### EXAMPLE 1
```
Update-CurrentUser -LastName "Smith"
```

Update the current user's lastname
