# Remove-EventCollection

## SYNOPSIS
Delete a collection of events

## SYNTAX

```
Remove-EventCollection
	[[-Device] <Object[]>]
	[[-Type] <String>]
	[[-FragmentType] <String>]
	[[-DateFrom] <String>]
	[[-DateTo] <String>]
	[-Revert]
	[[-ProcessingMode] <String>]
	[-Raw]
	[[-OutputFile] <String>]
	[-NoProxy]
	[[-Session] <String>]
	[[-TimeoutSec] <Double>]
	[-Force]
	[-WhatIf]
	[-Confirm]
	[<CommonParameters>]
```

## DESCRIPTION
Delete a collection of events by using a filter

## EXAMPLES

### EXAMPLE 1
```
Remove-EventCollection -Type my_CustomType -DateFrom "-10d"
```

Remove events with type 'my_CustomType' that were created in the last 10 days

### EXAMPLE 2
```
Remove-EventCollection -Device "{{ randomdevice }}"
```

Remove events from a device
