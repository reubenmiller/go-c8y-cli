# Get-AllTenantUsageSummaryStatistics

## SYNOPSIS
Get collection of tenant usage statistics summary

## SYNTAX

```
Get-AllTenantUsageSummaryStatistics
	[[-DateFrom] <String>]
	[[-DateTo] <String>]
	[-Raw]
	[[-OutputFile] <String>]
	[-NoProxy]
	[[-Session] <String>]
	[[-TimeoutSec] <Double>]
	[-WhatIf]
	[-Confirm]
	[<CommonParameters>]
```

{{ Fill in the Description }}

## EXAMPLES

### EXAMPLE 1
```
Get-AllTenantUsageSummaryStatistics
```

Get tenant summary statistics for all tenants

### EXAMPLE 2
```
Get-AllTenantUsageSummaryStatistics -DateFrom "-30d"
```

Get tenant summary statistics collection for the last 30 days

### EXAMPLE 3
```
Get-AllTenantUsageSummaryStatistics -DateFrom "-10d" -DateTo "-9d"
```

Get tenant summary statistics collection for the last 10 days, only return until the last 9 days
